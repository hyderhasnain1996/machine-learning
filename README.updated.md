# Machine Learning Lab

## About

This repository contains the Machine Learning Lab website built with Next.js. The site showcases the lab's research, publications, team, gallery, news, and contact information. It's organized using the Next.js App Router and Tailwind CSS for styling.

- Live demo: https://machine-learning-lab-three.vercel.app/
- Made by: Hasnain

## Getting Started

First, run the development server:

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load fonts.
